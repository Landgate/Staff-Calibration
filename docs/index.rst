Welcome to the Staff Calibration's technical documentation
==========================================================

Staff calibration is a web applicaiton for generating calibration parameters for levelling staves based on the Boya Staff Calibration Range. The application is developed using Django web framework (see, https://www.djangoproject.com/) and tested in Python 3.8.3 and Django 3.1.
